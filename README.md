# Cyperdeck

Welcome to my repository!<br> This is a fun little project inspired by the amazing cyberdecks I found on the internet.
It is nowhere near perfect, but the whole process like planning, sketching, modelling and building it was an incredibly fun experience. 
I learned a lot along the way and discovered how to turn an idea into something real.
